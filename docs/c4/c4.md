# C4 Model

A C4 model of the Recipe App for the project presentation, from system context down to the
components of the backend and frontend.

## Level 1: System Context

```puml
@startuml
!include <C4/C4_Context>

title System Context — Recipe App

Person(user, "Home Cook", "Manages and views cooking recipes")
System(app, "Recipe App", "Browse, search and manage recipes and ingredients")
System_Ext(weather, "Weather API", "Provides weather information")

Rel(user, app, "Uses", "HTTPS")
Rel(app, weather, "Requests weather information", "HTTPS")

SHOW_LEGEND()
@enduml
```

## Level 2: Container

```puml
@startuml
!include <C4/C4_Container>

title Container View — Recipe App

Person(user, "Home Cook", "Uses a web browser")

System_Boundary(app, "Recipe App") {
  Container(fe, "Frontend", "React 19 SPA, served by nginx", "Renders the UI; proxies /api to the backend")
  Container(be, "Backend", "NestJS REST API (Node 24)", "Recipes, ingredients and auth; validates input and enforces JWT auth")
  ContainerDb(db, "Database", "SQLite file", "Stores recipes, ingredients and users")
  Container(admin, "sqlite-web", "coleifer/sqlite-web", "Web UI to inspect the database (dev convenience)")
}

System_Ext(weather, "Weather API", "Provides current temperature and weather code (open-meteo.com)")

Rel(user, fe, "Visits", "HTTPS")
Rel(fe, be, "Calls", "REST /api, JSON, JWT bearer")
Rel(be, db, "Reads & writes", "SQL (file)")
Rel(admin, db, "Inspects", "SQL (file)")
Rel(be, weather, "Requests weather information", "HTTPS")

SHOW_LEGEND()
@enduml
```

## Level 3: Components — Backend

```puml
@startuml
!include <C4/C4_Component>

title Component View — Backend

Container(fe, "Frontend", "React SPA", "Sends REST requests")
ContainerDb(db, "Database", "SQLite", "Persists data")
System_Ext(weather, "Weather API", "Provides current temperature and weather code (open-meteo.com)")

Container_Boundary(be, "Backend (NestJS)") {
  Component(guard, "AuthGuard", "NestJS guard (APP_GUARD)", "Protects every route by default; @Public() opts out")
  Component(pipe, "ValidationPipe", "NestJS pipe", "Validates & whitelists request bodies against DTOs")

  Component(recipeCtrl, "Recipe Controller", "NestJS controller", "/api/recipes endpoints")
  Component(recipeSvc, "Recipe Service", "NestJS service", "Recipe business logic")
  Component(ingCtrl, "Ingredient Controller", "NestJS controller", "/api/recipes/:id/ingredients endpoints")
  Component(ingSvc, "Ingredient Service", "NestJS service", "Ingredient business logic")
  Component(authCtrl, "Auth Controller", "NestJS controller", "POST /api/auth/login")
  Component(authSvc, "Auth Service", "NestJS service", "Verifies credentials, issues JWTs")
  Component(weatherCtrl, "Weather Controller", "NestJS controller", "GET /api/weather (@Public)")
  Component(weatherSvc, "Weather Service", "NestJS service", "Fetches and maps weather data")
  Component(weatherRepo, "Weather Repository", "NestJS service + opossum circuit breaker", "Calls the external Weather API; falls back gracefully")
  Component(repo, "Repositories & Entities", "TypeORM", "Maps entities to the SQLite schema")
}

Rel(fe, guard, "Sends requests through", "REST /api, JWT bearer")
Rel(guard, recipeCtrl, "Allows")
Rel(guard, ingCtrl, "Allows")
Rel(guard, authCtrl, "Allows (@Public login)")
Rel(pipe, recipeCtrl, "Validates")
Rel(pipe, ingCtrl, "Validates")

Rel(recipeCtrl, recipeSvc, "Uses")
Rel(ingCtrl, ingSvc, "Uses")
Rel(authCtrl, authSvc, "Uses")

Rel(recipeSvc, repo, "Reads & writes")
Rel(ingSvc, repo, "Reads & writes")
Rel(authSvc, repo, "Reads users")
Rel(repo, db, "SQL", "better-sqlite3")
Rel(guard, weatherCtrl, "Allows (@Public weather)")
Rel(weatherCtrl, weatherSvc, "Uses")
Rel(weatherSvc, weatherRepo, "Uses")
Rel(weatherRepo, weather, "Fetches weather data", "HTTPS")

SHOW_LEGEND()
@enduml
```

## Level 3: Components — Frontend

```puml
@startuml
!include <C4/C4_Component>

title Component View — Frontend

Person(user, "Home Cook", "Uses the browser")
Container(be, "Backend", "NestJS REST API", "Serves /api")

Container_Boundary(fe, "Frontend (React SPA)") {
  Component(home, "Home Page", "React", "Lists & searches recipes (client-side filter)")
  Component(recipe, "Recipe Page", "React", "Recipe detail; scales ingredient amounts client-side")
  Component(dialogs, "Dialogs & Components", "React + MUI", "Reusable forms for recipes and ingredients")
  Component(auth, "AuthContext", "React context", "Holds the JWT in localStorage; login/logout state")
  Component(api, "api.ts", "API client", "Wraps /api calls; attaches Authorization: Bearer")
  Component(weatherComp, "Weather Component", "React", "Displays current temperature and weather icon")
}

Rel(user, home, "Browses")
Rel(user, recipe, "Views")
Rel(home, dialogs, "Opens")
Rel(recipe, dialogs, "Opens")
Rel(home, api, "Calls")
Rel(recipe, api, "Calls")
Rel(dialogs, api, "Calls")
Rel(api, auth, "Reads token from")
Rel(api, be, "Sends requests to", "REST /api, JWT bearer")
Rel(home, weatherComp, "Renders")
Rel(recipe, weatherComp, "Renders")
Rel(weatherComp, api, "Calls getWeather()")

SHOW_LEGEND()
@enduml
```
